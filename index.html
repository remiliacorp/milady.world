Installation Instructions


STEP 1: INSTALL JDK


1: You need a Java Development Kit (JDK) version 17. It's provided in the "JDK 17 Installer" folder.

1.1: Select the appropriate package for your operating system inside the 'JDK 17 Installer' folder. M1/M2 Mac users should use the 'M1_M2_mac' file, other Mac users the 'intel_mac' file.

(For Mac users, if you do not know whether you have an M1/M2 Mac or not, go to the Apple logo in the top bar, 'About This Mac' and you should see M1/M2 on the Overview if you have one)

1.2: Run the JDK installer and follow the installation wizard. You should not have to change any options.

1.3: You now should have Java 17 installed to play Minecraft with!


Step 2: SETUP MILADYCRAFT


2: Within the 'Miladycraft Launcher' folder, extract the archive appropriate to your operating system, named 'Miladycraft - [your operating system]-...'

2.1: IMPORTANT FOR MAC USERS, if your Mac Operating System is below Catalina (version 10.15), use the 'Legacy macOS' version. If you do not know your operating system, go to the Apple logo in the top bar, 'About This Mac' to check.


STEP 3: LAUNCH MILADYCRAFT


3: Go into the folder extracted from the archive, find and launch the Miladycraft Launcher application.

4: Select your language, click 'Next' at the bottom right of the window, then select your Java 17 installation. You should automatically see the filepath where you installed Java in Step 1, which should be labelled with Version 17.x.x

5. At the bottom of the Java setup window, change your memory so that 'Minimum memory allocation' is set to '3072 MiB' and 'Maximum memory allocation' is set to '4096 MiB'. Click 'Finish' at the bottom right of the window. (If you *know* your computer has more than 8 GB of RAM available, you can set your minimum allocation equal to maximum, i.e. 4096 MiB for a slight performance increase. Allocating more maximum memory is possible, but not recommended as Miladycraft has been successfully tested for the given amounts.)

6: Go to 'Settings' in the top bar of the PollyMC launcher program.

7: Select 'Java' from the left sidebar. You can review your memory allocations from Step 5 here if you are unsure. Go down to the 'JVM arguments' text box, and copypaste the text below, which will optimize your Java performance for Minecraft:

-XX:+UnlockExperimentalVMOptions -XX:+UnlockDiagnosticVMOptions -XX:+AlwaysActAsServerClassMachine -XX:+AlwaysPreTouch -XX:+DisableExplicitGC -XX:+UseNUMA -XX:-DontCompileHugeMethods -XX:MaxNodeLimit=240000 -XX:NodeLimitFudgeFactor=8000 -XX:+UseVectorCmov -XX:+PerfDisableSharedMem -XX:+UseFastUnorderedTimeStamps -XX:+UseCriticalJavaThreadPriority -XX:ThreadPriorityPolicy=1 -XX:AllocatePrefetchStyle=3 -XX:+UseG1GC -XX:MaxGCPauseMillis=37 -XX:+PerfDisableSharedMem -XX:G1HeapRegionSize=16M -XX:G1NewSizePercent=23 -XX:G1ReservePercent=20 -XX:SurvivorRatio=32 -XX:G1MixedGCCountTarget=3 -XX:G1HeapWastePercent=20 -XX:InitiatingHeapOccupancyPercent=10 -XX:G1RSetUpdatingPauseTimePercent=0 -XX:MaxTenuringThreshold=1 -XX:G1SATBBufferEnqueueingThresholdPercent=30 -XX:G1ConcMarkStepDurationMillis=5.0 -XX:G1ConcRSHotCardLimit=16 -XX:G1ConcRefinementServiceIntervalMillis=150 -XX:GCTimeRatio=99


7.1: If you *know* your computer has more than 8 GB of RAM available, you can use these instead:

-XX:+UnlockExperimentalVMOptions -XX:+UnlockDiagnosticVMOptions -XX:+AlwaysActAsServerClassMachine -XX:+AlwaysPreTouch -XX:+DisableExplicitGC -XX:+UseNUMA -XX:NmethodSweepActivity=1 -XX:ReservedCodeCacheSize=400M -XX:NonNMethodCodeHeapSize=12M -XX:ProfiledCodeHeapSize=194M -XX:NonProfiledCodeHeapSize=194M -XX:-DontCompileHugeMethods -XX:MaxNodeLimit=240000 -XX:NodeLimitFudgeFactor=8000 -XX:+UseVectorCmov -XX:+PerfDisableSharedMem -XX:+UseFastUnorderedTimeStamps -XX:+UseCriticalJavaThreadPriority -XX:ThreadPriorityPolicy=1 -XX:AllocatePrefetchStyle=3 -XX:+UseG1GC -XX:MaxGCPauseMillis=37 -XX:+PerfDisableSharedMem -XX:G1HeapRegionSize=16M -XX:G1NewSizePercent=23 -XX:G1ReservePercent=20 -XX:SurvivorRatio=32 -XX:G1MixedGCCountTarget=3 -XX:G1HeapWastePercent=20 -XX:InitiatingHeapOccupancyPercent=10 -XX:G1RSetUpdatingPauseTimePercent=0 -XX:MaxTenuringThreshold=1 -XX:G1SATBBufferEnqueueingThresholdPercent=30 -XX:G1ConcMarkStepDurationMillis=5.0 -XX:G1ConcRSHotCardLimit=16 -XX:G1ConcRefinementServiceIntervalMillis=150 -XX:GCTimeRatio=99


8: Select 'Accounts' from the left sidebar, and pick one of the 'Add' options on the right sidebar to add your account! You don't need a Mojang account to play on Miladycraft, so it is most convenient to select 'Add Offline' and pick the username of your choice. Do NOT select 'Allow long usernames'! You can pick any username you'd like, so long as it isn't already being used by someone else on the server. (You can set and/or change your skin ingame)

9: Close the Settings window, and select the 'Add Instance' window in the top bar from the main window.

10: Select 'Import from zip' in the left sidebar, select 'Browse' right to the URL field, and select 'Miladycraft 1.3.1.zip' from your package folder. This may take a while!

11: Close the New Instance window, and click on the newly created instance in the main launcher window to launch Minecraft.

12: Once Minecraft has finished loading, select 'Multiplayer.' You will now set a password for your account, which you can set yourself, or which can be randomly generated using the 'R' button next to the input field. You don't need to input your password again later, as it will be stored inside your PollyMC folder.

12.1: Back up your password by navigating to your Minecraft folder inside the launcher: in the main launcher window, go to the right sidebar and select 'Folder'. From the window now opened in your file explorer, navigate to '.minecraft', where the '.sl_password' file can be found. This is the file which stores your password, and you need it right here to play! Copy it and store the copy somewhere else safely, in case you have to reinstall Minecraft later, or want to play on another computer!

13: You are now ready to play! Select the Milady World multiplayer server from Minecraft's Multiplayer panel and have fun!
